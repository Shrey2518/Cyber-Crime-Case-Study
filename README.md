# Cyber-Crime-Case-Study
The 2017 NotPetya cyberattack was a highly destructive global event disguised as a ransomware campaign but functioned as a wiper malware designed for pure sabotage. It initially spread via a corrupted software update for the Ukrainian tax program M.E.Doc. Once inside a network, NotPetya became an unstoppable worm by combining the EternalBlue.
